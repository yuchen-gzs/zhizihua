    实时密钥生成程序操作说明
===============================================

【程序简介】
这是一个功能强大的实时密钥生成器，能够生成多种类型的安全密钥。
程序采用Java Swing图形界面，提供直观的操作体验和实时生成功能。

【主要功能】
1. 实时自动生成密钥（每2秒生成一次）
2. 支持8种不同类型的密钥生成
3. 可自定义密钥长度和字符集
4. 密钥历史记录管理
5. 一键复制到剪贴板
6. 导出密钥历史记录

【密钥类型】
1. 随机密码 - 可选择包含大写字母、小写字母、数字、特殊字符
2. 十六进制密钥 - 使用0-9和A-F字符
3. UUID - 标准UUID格式密钥
4. 时间戳密钥 - 基于当前时间戳的唯一密钥
5. 数字密钥 - 纯数字组成的密钥
6. Base64密钥 - Base64编码格式的密钥
7. 自定义格式 - 根据模式生成密钥（如：XXXX-XXXX-XXXX）
8. 带前缀密钥 - 带有自定义前缀的密钥

【操作方法】
1. 启动程序：运行 java KeyGenerator
2. 选择密钥类型：从下拉菜单选择所需的密钥类型
3. 设置参数：
   - 调整密钥长度（4-128位）
   - 选择字符集（大写、小写、数字、特殊字符）
   - 输入自定义模式（如需要）
   - 设置前缀（如需要）
4. 自动生成：勾选"自动生成"复选框，程序将每2秒生成一次密钥
5. 手动生成：点击"立即生成"按钮手动生成密钥
6. 复制密钥：点击"复制密钥"按钮复制最新生成的密钥
7. 查看历史：在右侧历史列表中查看之前生成的密钥
8. 复制历史：双击历史列表中的任意密钥即可复制
9. 保存历史：点击"保存历史"按钮将历史记录导出为文本文件
10. 清空显示：点击"清空显示"按钮清空主显示区域

【自定义模式说明】
在自定义格式模式下，可以使用以下占位符：
- X：大写字母
- x：小写字母
- 9：数字
- H：十六进制字符
- 其他字符：原样保留

示例模式：
- "XXXX-XXXX-XXXX"：生成如 "ABCD-EFGH-IJKL" 的密钥
- "xxx999"：生成如 "abc123" 的密钥
- "KEY-HHHH-9999"：生成如 "KEY-A1B2-3456" 的密钥

【安全特性】
1. 使用 SecureRandom 类确保密钥的随机性和安全性
2. 支持多种字符集组合，提高密钥强度
3. 实时生成，避免密钥重复使用
4. 本地生成，不依赖网络，保证安全性

【系统要求】
- Java 8 或更高版本
- 支持 Swing 图形界面的操作系统
- 至少 50MB 可用内存

【快捷键】
- Ctrl+C：复制选中的密钥
- F5：立即生成新密钥
- Esc：退出程序

【注意事项】
1. 生成的密钥仅在程序运行期间保存在内存中
2. 关闭程序后历史记录将丢失，请及时保存重要密钥
3. 建议定期更换密钥以确保安全性
4. 特殊字符可能在某些系统中显示异常，属正常现象
5. 密钥长度过长可能影响程序性能

【故障排除】
1. 程序无法启动：检查Java版本是否符合要求
2. 界面显示异常：尝试更新显卡驱动
3. 复制功能失效：检查系统剪贴板权限
4. 保存失败：检查文件写入权限
